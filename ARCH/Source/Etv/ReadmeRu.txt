		ПОСЛЕДНИЕ ИЗМЕНЕНИЯ 
		===================

Версия 3.5
- Совместимость с Delphi 6 и Delphi 7.
- Протестировано с ADO, Clientdataset, DBISAM 3.x и другими датасетами. 
- Добавлена возможность переноса слов при печати. 
- TEtvCloneRecord.Caption. TEtvCloneRecord.Subdatasets.Caption 
- Исправлена проблема с печатью страниц в EtvPrint. 
- Новые функции в юните EtvDBFun. 
- Исправлены ошибки. 

Версия 3.4
- Совместимость с различными BDE alternatives.
- TEtvDBGrid: property Lookups для определения lookup полей прямо в гриде;
  переделан drag-and-drop; добавлено попап меню для lookupСombo.   
- Lookup combo box: возможность переключения на другой справочный dataset; 
  фильтры по нескольким колонкам без определения события; возможность 
  ограничения списка при поиске, возможность присвоения значения при 
  нажатии Tab, автозаполнение при наборе, серый текст при ReadOnly; 
  новые функции во всплывающем меню.
- Включение/отключение акселерации (перехода по Enter) для грида и др. 
  элементов управления.
- TEtvDBText, добавлены свойства LookupDataSet, LookupKeyField.
- EtvConst unit: добавлены различные значения по умолчанию.
- TEtvFilter:  Возможность выбора записей без подзаписей; настройки на 
  различные dataset'ы.
- TEtvRecordScrollBox - компонент для создания окна редактирования одной 
  записи.

Версия 3.3
- Все юниты переработаны, библиотека разделена на базовый пакет и несколько 
  частей, каждая часть может работать независимо;
- Etv Library протестировано с Delphi 5.0 Update Pack 1. 
- Свойство FieldQuotes добавлено в компоненты TEtvFilter и TEtvSortingCombo;
- Свойства Caption и ExternalInfo добавлены в компонент TEtvFilter и его  
  свойство SubDataSets;
- Свойство NotUseOldFilter добавлено в компонент TEtvFilter; 
- Параметр Quotes добавлен в процедуры AllFieldNames, SortingToDataSet и
  SortingToSQL;
- Значение AutoWidth добавлено в свойство Options формы TFormDB;
- Исправлено несколько ошибок. 

Версия 3.2
- TEtvDBLookupCombo может менять сортировку LookupDataSet
- Popup menu компонента TEtvDBLookupCombo
- Некоторые ошибки исправлены

Версия 3.1
- поддержка Delphi 5
- возможность работы без BDE, смотри файл Etv.inc
- TEtvFilter работает с любыми датасетами
- Механизм вызова форм редактирования работает с любыми датасетами - 
  событие OnEditData у TEtvDBLookupCombo и TEtvLookField
- Свойство TotalFont у TEtvDBGrid
- Свойство StoreLookupData в TEtvLookField для совместимости с различными 
  датасетами. Если TEtvLookField не работает с Вашими датасетами при 
  StoreLookupData=true, устанавливайте свойство в false. Например, это
  нужно для ADO датасетов. Иначе, для ускорения, устанавливайте это 
  свойство в true. Например, StoreLookupData=true будет работать со всеми 
  наследниками TBDEDataset. По умолчанию StoreLookupData=false

     1.1. Инсталляция для Delphi 3
   Если у Вас установлена одна из предыдущих версий Etv Library, 
деинсталлируйте ее и удалите все старые скомпилированные пакеты.
   Загрузите Delphi. Закройте текущий проект. Скомпилируйте run-time 
пакет RtlEtv3 и поместите полученные RtlEtv3.dcp и RtlEtv3.dpl файлы 
в каталог, доступный в системе (указанный в команде Path). 
Затем инсталируйте дизайновый пакет DclEtv3. 

После этого ОБЯЗАТЕЛЬНО перезагрузите Delphi.

Возможная проблема! Если у Вас установлена библитотека RX Library, 
на время инсталляции необходимо убрать переопределение юнита VerInfo. 
Т.е. Tools - Environment options - Library - Unit Aliases - строка 
VerInfo=RxVerInf

   2. Файл помощи

Borland Delphi 6-7 требует, чтобы все ключевые слова файла помощи были 
добавлены в d7.als файл (d6.als для Delphi 6).  Если вы хотите иметь полностью

с нашего сайта и воспользоваться ей.

		КРАТКОЕ ОПИСАНИЕ
		================
  
	Компоненты для поддержки лукапной информации. 

- Определение нескольких полей в LookupResultField наследника TField'а;
- Поиск с отображением вводимой строки; переключение и отображение столбца 
  поиска/редактирования; 
- Многоуровневые лукапы с фиксированным числом уровней и деревья;
- Наложение фильтров на список вывода лукапа; 
- Сохранение значений полей из LookupDataSet'а для дальнейшего использования 
  в программе (из свойства LookupResultField и из др. спец. свойств); 
- Компоненты для отображения сохраненных значений;
- Отображение нескольких полей в отдельном лукапе и в гриде; 
- Прямое редактирование KeyField в лукапе и гриде; 
- Вызов формы редактирования LookupDataSet'а; 
- Определение TFont для любой строки лукапа;
- Строка заголовков полей для списка вывода лукапа; 
- Автоматический выбор вводимого столбца (для полей "Число;Текст");
- Возможность внесения значений, отсутствующих в LookupDataSet'е;
- Смена сортировки LookupDataSet'а.

	Грид

- Многострочные заголовки полей;  
- Определение Font и Color для любой ячейки;
- Определение видимости и очередности полей пользователем
- Вывод информации на печать в текстовом и графическом режиме 
  с возможностью разбивки на страницы, определения параметров страницы, 
  шрифтов, межстрочного интервала и др.; вывод в файл
- Смена раскладки клавиатуры; 
- Cтрока итогов для показа сумм по столбцам, количества записей, 
  любой другой информации; 
- Лукапные поля (см. выше); 
- Поля фиксированных списков;
- Клонирование записи
- Генерация окна редактирования для одной записи
- Возможность подстановки своих элементов управления 
  (например, для редактирования даты)

	Выборки, сортировка и поиск

- Компонент TEtvFilter позволяет конечному пользователю определять и 
  ПОЛУЧАТЬ выборки как по отдельной таблице так и по группе таблиц 
  связанных отношением Master-Detail.
  В зависимости от установок происходит наложение фильтра на датасет 
  или генерация SQL запроса. Во втором случае Ваш датасет будет заменем 
  на другой на время установки выборки. Вы можете определять класс 
  генерируемого датасета, либо указать на конкретный DataSet.
  Параметры выборок могут быть сохранены в файле или в базе данных. 

- Компонент TEtvDBSortingCombo - смена cортировки датасета, в том числе 
  и для датасетов типа TQuery (имеющих свойство SQL). Список возможных 
  сортировок может быть определен пользователем или равняться списку 
  индексов для датасетов типа TTable

- Компонент TEtvFindDlg - поиск записи по полям сортировки, в том числе 
  и для датасетов типа TQuery (имеющих свойство SQL).

	Фиксированные списки

Компоненты для работы/отображения небольших фиксированных списков, 
в соответствие которым в базе ставятся поля Smallint. 
Например, Маленький/Средний/Большой, в базе 0/1/2. 

	Подстановка своих элементов управления в библиотеку.

Предусмотрена возможность определения и подстановки своих наследников 
от TControl. Например, авторы в своих проектах успешно используют 
TDateEdit от RX. Ваши элементы управления будут использоваться при 
определении выборок, поиска, при автоматической генерации компонентов 
редактирования/просмотра одной записи и, кроме того, можно определять 
элементы редактирования для использования в гриде.

	Другие компоненты и механизмы
 
- Механизм вызова форм редактирования
- Элементы управления для полей данных
- Popup menu элементов управления
- Popup menu компонентов в дизайнере
- TEtvTable, TEtvQuery - редактирование данных в дизайнере; 
  поддержка лукапных полей; Popup меню в дизайнере; проверка 
  MasterSource при вставке записи, автоинкрементные поля и др.
- TFormDB и TFormBase - вариант базовой формы для отображения данных -
  автоматическая генерация страницы редактирования одной записи, 
  определение выборок, сортировка, поиск, обновление информации
- TEtvPageControl+TEtvTabSheet - отключение данных на неактивных страницах 
  для ускорения работы
- TEtvApp object - открытие/обновление данных
- TEtvDMInfo component - информация о компонентах датамодуля; 
  открытие/закрытие всех датасетов модуля
- TEtvPrinter - Объект для вывода информации на принтер в текстовом и 
  графическом виде; диалог для определения различных параметров печати;
- TEtvRecordCloner - клонирование записей для группы датасетов связанных 
  отношением Master-Detail (1:N)
- TEtvDBRichEdit - добавлено Popup Menu с различными функциями, сквозной 
  поиск и замена, т.е. с переходом от записи к записи
- Другие компоненты и функции
