		Общие команды управления формой с клавиатуры.
<Ctrl+F>	- вызов окна поиска для активной таблицы.
<Сtrl+K>	- активизирование открытого списка ключей активной таблицы.
<Alt+T>		- выкл./вкл. панели инструментов активной формы.
<Ctrl+A>	- возврат значения в элемент "список выбора" из предыдущей формы.
<Ctrl+Home/
Ctrl+End>	- на первую/последнюю запись.		

		      Краткий перечень клавиатурных команд для ЭКРАННОЙ ФОРМЫ:

<Enter/
Shift+Enter>	- на следующее/предыдущее поле.
<PgDown/PgUp>	- на следующую/предыдущую запись.
<Ctrl+~>	- циклический переход между нижней и верхней частями формы и 
		  обратно.
<Ctrl+"+"/
Ctrl+"-">	- вставить/удалить запись.

  Команды клавиатуры для элемента управления "СПИСОК ВЫБОРА" 
      ("ВЫПАДАЮЩЕЕ МЕНЮ") (расширенный вариант).

<Ctrl+Enter>	- открыть список/закрыть список с присвоением значения.
<Esc>		- закрыть открытый список без присвоения значения.
<F2,Del,Enter>	- очистка значения (установка пустого значения).
<Ctrl+Shift+Z>	- циклическая смена порядка следования полей в списке.
<Ввод значения>	- в закрытом состоянии можно ввести код, если точно известно его
		  значение. Затем нажимаем <Enter>.
<Ввод значения>	- в открытом состоянии будет осуществляться поиск строки списка,
		  начало которой точно совпадает с введенным значением.
<Ввод значения>,
  <Ctrl+F>	- в открытом состоянии: формирование подмножества полного списка,
		  состоящего из строк, содержащих в себе
		  подстроку=<введенному значению> (поиск по контексту).
<Вверх-Вниз>	- в закрытом и открытом состоянии: перемещение по списку.
<Enter>		- в открытом состоянии присвоение значения с переходом на 
		  следующее поле или переход на следующий уровень в 
		  многоуровневых списках выбора.
<Home/End>	- в открытом состоянии переход в начало/конец списка.
<Ctrl+S>	- перейти в соответствующую списку форму(справочник)
		  для корректировки(просмотра).

	Команды клавиатуры для управления элементом "ТАБЛИЦА"
<Стрелки>	- в режиме ПРОСМОТРА: перемещение по ячейкам таблицы.
<PgUp/PgDown>	- в режиме ПРОСМОТРА: перемещение на страницу вверх/вниз.
<Home/End>	- а) в режиме ПРОСМОТРА: в начало/конец строки таблицы.
		  б) в режиме РЕДАКТИРОВАНИЯ: в начало/конец строки ячейки.
<Ctrl+Home/
Ctrl+End>	- в начало/конец таблицы.
<Ins>		- вставка новой записи.
<Ctrl+End>,
<Стрелка вниз>  - добавление новой записи в конце таблицы.
<Ctrl+Del>	- удаление текущей записи.
<Enter>		- вход в режим РЕДАКТИРОВАНИЯ текущей ячейки.
<Esc>		- а) в режиме РЕДАКТИРОВАНИЯ ячейки - выход в режим ПРОСМОТРА
		  с возвратом отредактированного значения к старому;
		  б) в режиме ПРОСМОТРА - отмена всех отредактированных значений
		  по текущей строке и возврат к старым значениям.
